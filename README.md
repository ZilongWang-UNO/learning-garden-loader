# SR1 Lesson Loader

A small, framework-free prototype for an educational video loading experience.

The current version establishes the surrounding lesson page:

- SR1-inspired responsive navigation
- Course progress and lesson outline
- Embedded YouTube lesson
- Keyboard-friendly controls and mobile menu

The 20-second educational loading experience will be added inside the video
area next.

## Run locally

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.

## Project structure

```text
index.html   Page structure and lesson content
styles.css   Layout, visual system, and responsive styles
script.js    Small interface interactions
```

No package installation or build step is required.
